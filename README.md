# 🏏 IPL Data Analysis — Exploratory Data Analysis

## 📌 Overview
Exploratory Data Analysis (EDA) on the Indian Premier League (IPL) dataset
using Python. This project uncovers trends in team performance, player 
statistics, toss impact, and season-wise match data.

## 📊 Key Insights
- Most successful teams by total wins
- Toss impact on match results
- Season-wise match trends (2008–2023)
- Top Player of the Match award winners
- Most used venues across IPL history

## 🛠️ Tools & Libraries Used
- Python 3
- Pandas
- Matplotlib
- Jupyter Notebook

## 📁 Files
| File | Description |
|------|-------------|
| `eda-ipl-dataset.ipynb` | Main EDA notebook with all analysis and charts |

## 🚀 How to Run
1. Clone this repository
2. Open `eda-ipl-dataset.ipynb` in Jupyter Notebook or Google Colab
3. Run all cells
